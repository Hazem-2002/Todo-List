# 📝 Todo List App

A modern and fully responsive Todo List application built using **React.js** and **Material UI (MUI)**. The app provides a smooth and interactive experience for managing daily tasks with persistent data storage and efficient state management.

## ✨ Features

* ➕ Add new tasks (Title + Details) using a Dialog
* ✏️ Edit existing tasks easily
* 🗑️ Delete tasks with confirmation dialog
* ✅ Mark tasks as completed with visual feedback (line-through)
* 🔄 Toggle task status (Completed / Uncompleted)
* 🔔 Snackbar notifications for all actions (Add / Edit / Delete)
* 📂 Task filtering using Tabs:

  * All Tasks
  * Completed Tasks
  * Uncompleted Tasks
* 💾 Data persistence using **localStorage**
* 📱 Fully responsive design
* 🌍 Arabic UI (with future support for English)

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Material UI (MUI)
* 🧠 React Hooks:

  * useState
  * useReducer
  * useContext
  * useMemo

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Hazem-2002/Todo-List-App.git
```

2. Navigate to the project folder:

```bash
cd Todo-List-App
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
